# VAT Invoice Extractor

**Phần mềm trích xuất hóa đơn VAT điện tử — XML sang Excel**

Tự động hóa việc đọc và chuyển đổi hóa đơn VAT điện tử từ định dạng XML sang bảng tính Excel. Hỗ trợ hóa đơn từ MISA, Fast, VNPT, Viettel, FPT theo Thông tư 78/2021/TT-BTC.
